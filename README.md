# Sales Dashboard (Power BI)

## Project Overview

This project presents a Sales Performance Dashboard built using Power BI. It provides insights into sales, profit, and customer trends using the Superstore dataset.

## Key Features

* Total Sales, Profit, and Orders KPIs
* Profit Margin analysis
* Sales trend over time
* Region-wise and category-wise performance
* Top-selling products
* Interactive slicers for filtering

## Key Insights

* Technology category generates highest sales
* West region contributes the most revenue
* Sales peak observed in Q4 months

## Tools Used

* Power BI
* Excel / CSV dataset
